# Competition Project

This project uses TypeScript + Webpack with a local mock HTTP server.

## Structure

- `data/weather-data.json`
- `html/*.html`
- `src/index.ts`
- `css/styles.css`
- `mock-server.js`
- `webpack.config.js`

## Commands

- `npm run build`
- `npm run dev -- --port 8081`
- `npm run mock`
